# AR IP 夥伴 — 配飾視覺風格指南

## STYLE_BASE（wrapper 讀第一個 code block）


## 用途

給 AR 角色（麻糬兔/熊布丁/啾啾，粉彩圓潤軟糖風）穿戴的配飾。
每張都是「單一配飾、純白背景、正面」，生成後會做白背去除轉透明 webp，以 2.5D 看板貼在 3D 角色身上。

## 色系

- 主色：粉紅 #FF7FA5、奶油白 #FFF3F6、鵝黃 #FFE066、焦糖 #C89F6E
- 禁：寫實場景、金屬冷色、暗黑風、任何背景環境
